# WaterWarmth GitHub Pages site

This folder is ready to publish as a static GitHub Pages site.

## Required files

- `index.html` — main page
- `pilot-map-embed.html` — embedded Leaflet map
- `sites-data.js` — map data
- `.nojekyll` — tells GitHub Pages to serve the files as-is

## Publish

1. Create a GitHub repository.
2. Upload all files in this folder to the repository root.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select **main** and **/(root)**, then click **Save**.

For a repository named `my-site`, the URL will normally be:

`https://YOUR-USERNAME.github.io/my-site/`
